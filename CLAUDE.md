# CLAUDE.md

The instructions for this repo are tool-neutral and live in **`AGENTS.md`** — read
that. It covers the one core rule, the three namespaces, the `common.js` toolkit,
the anchoring notes, house style, and the everyday workflows. Detailed
per-language rules are in `rules/`; sharp edges already hit are in `todo/diary.md`.

Claude Code also exposes the workflows as slash commands via `.claude/skills/`
(`/new-page`, `/add-icon`, `/design`, `/review`, `/diary`, `/up`, `/down`).
