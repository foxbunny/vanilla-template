Spacing

One relative scale — every gap and pad is a --space-* token, never a hardcoded length. The bar shows each token's real size.

--space-10.25rem
--space-20.5rem
--space-31rem
--space-41.5rem
--space-52.5rem
--space-64em
--space-76em
--space-88em
--space-912em
--space-1016em

Small steps (1–5) are rem, fixed to the root size, for gaps and padding. Large steps (6–10) are em, so big whitespace scales with the local text. The .flow utility stacks content using these — its gap defaults to 1em, and a container sets --flow-space to any of them to retune.

Side nav

A ul of icon + label links. The current page isn't a link — it's a plain <span>, filled like the primary button. Being the one item you can't click marks it as current, so it needs no aria-current.

Cards

A framed surface. The frame adds no padding — build the inside from optional parts: a full-width .card-head (one tone deeper, with a bottom line), a .card-media that runs edge to edge, and a padded .card-body. Mix them as the content needs.
